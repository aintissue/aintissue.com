Frequently Asked Questions

Here are some frequently asked questions (FAQs) that we've gathered for you:

1. What's the quickest way to test how this works?

Join our Telegram group (AintIssue) and then open our Telegram bot (AintIssueBot) and start it. Send message to the bot and you will see it in the group. Reply to it in the group and you will receive it through the bot.

This is how our user support works in order to be transparent and to demo how it works, however, most projects set it up in their private group for customer support agents, with their own bot.

2. Can Aint Issue be self-hosted?

Yes, it is an Open Source project, you can find it on our GitHub profile and install it on your own server, set it up and use it completely free. But you can also use it for free just by signing up on our app.

3. What programming language is Aint Issue written in?

Most of important components are written in Golang, components like Telegram bot itself.

4. How does Aint Issue work?

User sends a message to our bot (or your bot) and that message is forwarded into your private Telegram group with user support agents. When they reply to that message in the group, the reply gets forwarded back to the user.
