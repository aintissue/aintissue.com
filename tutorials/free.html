Aint Issue Telegram Bot Free Setup Guide

Welcome to the Aint Issue Bot setup guide! This tutorial will quickly show you how to create a free customer support bot. Let's start! Need help? Just ask.

To set it up, follow these simple steps:

  • Add @AintIssueBot to your group
  • Run /register command in the group
  • Use link on your website as customer support contact point

First step is simple, you add the bot in your project's public or private support group and make it admin so it can do actions when messages are sent in the group.

In the second step you need to run /register command in the group. If group is not public, you need to run it as /register ProjectName where ProjectName is the name of your project. If it is public group, you can just run /register and it will use group's name as project name.

Third step is to use the link provided in the response of /register command as a contact point for your users or customers. When they open that link, their messages will be forwarded to your user support group. To reply to them, reply to message in the group.

Organize user support for your startup, SaaS, Open Source project or a company with this convenient and simple Telegram bot.
